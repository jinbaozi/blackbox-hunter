# CWE Pattern Reference

Common binary-analysis mappings:

- CWE-119 / CWE-120: unsafe fixed-size memory copies and unchecked buffer writes.
- CWE-125: reads indexed by untrusted length without bounds checks.
- CWE-190: integer overflow before allocation or copy length calculation.
- CWE-22: archive or path parsing that permits traversal.
- CWE-78: command construction from external input.
- CWE-798: embedded credentials or static API keys.
